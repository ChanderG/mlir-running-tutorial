## Aim

Alex's tutorial is an amazing starting resource to MLIR. But, after looking at and compiling MLIR programs, how to run them? That is where this tutorial comes in, in 4 parts:

1. How to run a MLIR program and obtain the results?
2. How to feed in inputs to a MLIR program and run it?
3. Comparing outputs: Is our output accurater after all the passes?
3. Comparing runtimes: How fast is our optimized function?
